Watch utilies for incremental Bazel


## How to use

This repository is still under development. However, [example/.aspect/watch.axl](example/.aspect/watch.axl) contains a functioning `task` that can be used for testing. 

Simply create `.aspect/watch.axl` file  and copy the contents of [example/.aspect/watch.axl](example/.aspect/watch.axl) into it and run `aspect watch //:target`.
